## キャラクターデザイナーの仕事

### 全体の役割

```
キャラクターデザイナーの仕事
├── 脚本からキャラクターのイメージを読み取る
├── ビジュアルデザインを提案
├── 画像生成AI（Qwen Image）で画像を生成
├── 複数のバリエーションを作成
└── キャラクターシートを完成させる
```

### キャラクターデザインのプロセス

```
1. 脚本・監督の指示を確認
   └→ キャラクターの性格、役割、時代背景を理解

2. デザインコンセプトを練る
   └→ 髪型、服装、表情、配色などを決定

3. プロンプトを作成
   └→ Qwen Image に最適なプロンプトを設計

4. 画像を生成
   └→ Qwen Image で画像を生成

5. 確認・調整
   └→ 複数のバリエーションを作成し、選定

6. キャラクターシートに記録
   └→ プロンプト、画像を記録
```

## プロンプト設計の指針

### 基本的なプロンプト構造

```
[基本描写] + [詳細な特徴] + [スタイル] + [品質指定]
```

### よいプロンプトの条件

1. **具体的である**
   - 「髪が長い」→「肩まである黒髪で、前髪は少し目にかかる」
   - 「美しい」→「彫刻のような美しさ、繊細な特徴」

2. **感情を含める**
   - 「悲しそうな表情」→「瞳に涙を浮かべ、口元を引き絞った悲しげな表情」

3. **スタイルを指定する**
   - 「アニメ風」「リアル」「水彩画」「油絵」など

4. **品質を指定する**
   - 「高品質」「詳細」「8K」「プロフェッショナル」など

### プロンプトの例

```
# 基本プロンプト
"若い女性、20代、黒髪のショートカット、鋭い瞳、
暗い雰囲気のコート、複雑な表情、悲しみと決意が同居、
アニメスタイル、高品質、詳細"

# 表情バリエーション
"同じキャラクター、微笑んでいる、優しい表情"
"同じキャラクター、怒り、眉をひそめた鋭い表情"

# ポーズバリエーション
"同じキャラクター、全身、立っているポーズ、雨の中に立つ"
"同じキャラクター、横顔、夜空を見上げている"
```

## Qwen Image の使用方法

### 基本的な使い方

```bash
# 画像生成
node .claude/skills/fal-ai/scripts/t2i-qwen-image-2512.ts \
  "{プロンプト}" \
  --size landscape_16_9 \
  --project {プロジェクト名}
```

### オプション

| オプション | 説明 | 値 |
|-----------|------|-----|
| `--size` | 画像サイズ | square, portrait_4_3, portrait_16_9, landscape_4_3, landscape_16_9 |
| `--format` | 出力形式 | jpeg, png, webp |
| `--num-images` | 生成枚数 | 1-4 |

## デザインの指針

### キャラクター性を表現する

1. **性格を外見に反映**
   - 明るい性格→明るい色、柔らかい表情
   - 暗い性格→暗い色、鋭い表情

2. **役割を服装で表現**
   - 主人公→目立つ特徴、象徴的な服装
   - 脇役→地味だが個性的な要素

3. **時代背景を反映**
   - 現代→現代的な服装
   - 未来→SF的な衣装
   - 過去→時代に合った服装

### 統一感を保つ

- 同じキャラクターのバリエーションは特徴を統一
- 作品全体のキャラクターでスタイルを統一
- 配色に一貫性を持たせる
