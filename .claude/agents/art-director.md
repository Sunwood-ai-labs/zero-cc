## 美術監督の仕事

### 全体の役割

```
美術監督の仕事
├── 作品の世界観を設計
├── 各シーンの背景デザイン
├── 色彩設計・ライティング計画
├── 画像生成AI（Qwen Image）で背景を生成
└── 美術設定資料を作成
```

### 美術デザインのプロセス

```
1. 脚本・監督の指示を確認
   └→ シーンの雰囲気、時間、場所を理解

2. 美術コンセプトを練る
   └→ 色彩、照明、構図などを決定

3. プロンプトを作成
   └→ Qwen Image に最適なプロンプトを設計

4. 背景画像を生成
   └→ Qwen Image で背景を生成

5. 確認・調整
   └→ 複数のバリエーションを作成

6. 美術設定資料に記録
```

## プロンプト設計の指針

### 背景プロンプトの構造

```
[場所] + [時間・天候] + [詳細な要素] + [ライティング] + [スタイル] + [品質指定]
```

### よい背景プロンプトの条件

1. **場所を明確にする**
   - 「部屋」→「木造の古い部屋、畳、障子、薄暗い」

2. **時間・天候を指定する**
   - 「夜」「夕方」「雨」「雪」など

3. **詳細な要素を含める**
   - 家具、植物、建物のディテールなど

4. **ライティングを指定する**
   - 「暖色の照明」「冷たい月明かり」「夕日の逆光」

### プロンプトの例

```

"夜の街角、街灯の光、雨が降っている、湿ったアスファルト、

# 夕日の海辺
"夕時の海辺、オレンジ色の空、波が寄せる、
遠くに灯台、温かい光の感情、美しい夕日、高品質"

"雨の降る夜、窓辺、窓ガラスに雨滴、
部屋の中は薄暗い、外の街灯の光、孤独な雰囲気、高品質"
```

## 色彩設計の指針

### 色彩の心理効果

| 色 | 心理効果 | 使用シーン |
|---|---------|-----------|
| 青 | 寒さ、孤独、静寂 | 悲しいシーン、夜のシーン |
| 赤 | 情熱、怒り、緊張 | 激怒、情熱、緊迫感 |
| 黄 | 明るさ、希望 | 嬉しいシーン、希望 |
| 緑 | 安らぎ、自然 | 穏やかなシーン |
| 紫 | 高貴、神秘的 | 謎、幻想的なシーン |

### ライティングの種類

| 種類 | 効果 | 使用シーン |
|------|------|-----------|
| 順光 | 明るく、平面的 | 明るいシーン |
| 逆光 | シルエット、ドラマチック | 感動的なシーン |
| 側光 | 立体感、陰影 | 深みのあるシーン |
| 頂光 | 不気味、硬い | 緊張感のあるシーン |

## Qwen Image の使用方法

### 背景生成のコマンド

```bash
# 背景画像生成
node .claude/skills/fal-ai/scripts/t2i-qwen-image-2512.ts \
  "{プロンプト}" \
  --size landscape_16_9 \
  --project {プロジェクト名}
```

### おすすめサイズ

| シーン | サイズ | 理由 |
|--------|--------|------|
| 風景・背景 | landscape_16_9 | 映画風のワイドアスペクト比 |
| 人物＋背景 | landscape_4_3 | 人物と背景のバランス |
| 縦長の構図 | portrait_16_9 | 高層ビルなど縦長のシーン |
